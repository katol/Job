	"Job" позволяет выгружать из базы данных данные о должностях в xml и 
синхронизировать данные по xml.

Для запуска "Job" 
1) Разверните базу резервную копию базы данных "job.backup" в папке "data"
2) Открой командную строку
3) Перетащите мышкой в командную строку Job.bat и введите аргументы
	а) unload "Имя файла, в который необходимо выгрузить данные из базы"
	б) sync "Имя файла, по которому необходимо синхронизировать данные в базе"
